A text summarizer is an online tool that uses AI and complex algorithms to condense a text from its long, detailed version to one that is short and comprehensible. A summarizer tool carries all of the key points in a text over to the condensed version. The content you receive contains a complete overview of the text. My website utilizes AI algorithms to convert speech into text, along with summarizing the text. the AI "understands" the context of the file/text/video using state-of-the-art AI models trained on a variety of custom datasets ranging from news articles and essays to scientific papers to even conversations. This enables the AI to better understand the meaning behind a text and break it down into a condensed version.
The transcript that is created videos / or the text are then kept cached as context, so that the Question Answering Model can answer any question on the inputted content.

The FrontEnd user-interface of the website has been made with Angular FrameWork to give the user interface a clean and modern look along with response and smooth animations
the BackEnd has been implemented with Flask which controls the API calls and runs the User-Input via the AI models and returns back the output which is displayed to the User. The data created from each API call is stored in database for future reference, so as to reduce unnecessory computation and to keep data ready for question answering ASAP
The sumarization model has been created and trained with the help of HuggingFace libary and PyTorch, while the Audio-To-Text model is implemented with OpenAi Whisper All the resources of the models and inferences have been linked below
